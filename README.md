## Application Web de Gestion Médicale (Élèves & Médecins)
## Description

Cette application web vise à simplifier la gestion des rendez-vous médicaux entre les élèves et les médecins, en offrant une plateforme sécurisée, conviviale et centralisée.
Elle permet une gestion efficace des consultations, des dossiers médicaux et des ordonnances, tout en facilitant la communication entre les deux parties.

## Objectifs du projet

Digitaliser la prise de rendez-vous médicaux

Centraliser les dossiers médicaux des élèves

Faciliter le suivi des consultations et des ordonnances

Offrir une interface simple et sécurisée pour les élèves et les médecins

##  Fonctionnalités – Médecin

Gérer les rendez-vous (Aujourd’hui / À venir)

Créer des consultations (analyses, état de santé, date, remarques)

Créer et gérer les ordonnances

Consulter l’historique des consultations de chaque élève

Accéder, modifier et consulter les dossiers médicaux des élèves

Ajouter des rendez-vous liés aux consultations

Rechercher les informations personnelles des élèves

Modifier ses informations personnelles

Supprimer son compte

##  Fonctionnalités – Élève

Gérer son agenda de rendez-vous (Aujourd’hui / À venir)

Prendre des rendez-vous en ligne

Consulter l’historique de ses consultations

Voir les ordonnances associées à chaque consultation

Imprimer une ordonnance

Accéder à son dossier médical personnel

Modifier ses informations personnelles

Supprimer son compte

## 🛠️ Technologies utilisées

Front-end : HTML , CSS

Back-end : Python

Base de données : MySQL
